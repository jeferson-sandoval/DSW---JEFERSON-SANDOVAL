Conceptos 02

¿Qué es GIT?

Git, es un sistema de control de versiones de código abierto, que permite controlar versiones para trabajar de una manera simple y optima cuando estamos desarrollando software. Ya que podemos controlar todos los cambios que se hacen en nuestra aplicación o código. Teniendo así el control absoluto de todo lo que pasa en el código, pudiendo volver atrás y abrir diferentes ramas de desarrollo.
GIT permite crear repositorios locales en todos los sistemas operativos más comunes, Windows, Linux o Mac.

¿Qué es GITHUB?

GITHUB es un proyecto comercial que tiene como función el alojamiento de repositorios de software gestionados por el sistema de control de versiones Git. Por tanto, Git es algo más general que nos sirve para controlar el estado de un desarrollo a lo largo del tiempo, mientras que GitHub es algo más particular: un sitio web que usa Git para ofrecer a la comunidad de desarrolladores repositorios de software. En definitiva, GitHub es un sitio web pensado para hacer posible el compartir el código de una manera fácil y al mismo tiempo darle popularidad a la herramienta de control de versiones en sí, que es Git.
