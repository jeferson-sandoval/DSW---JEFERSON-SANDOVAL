GIT y GITHUB

Introducción

Esta página contiene definiciones puntuales relacionadas con GIT Y GITHUB que es una herramienta de gran importancia para desarrolladores de sitios web, ya que es una herramienta informática de fácil manejo y arroja excelentes resultados.

Objetivo general del sitio:

  • Comprender las definiciones y funciones propias de los sistemas que controlan los cambios en el desarrollo de cualquier tipo de software, permitiendo conocer el estado actual de un proyecto.

Objetivo especifico:

  • Comprender las definiciones y características de GIT y GITHUB en el entorno funcional del sistema en cada caso individual.
  • Conocer las relaciones que se generan en la ejecución de ambos sistemas, que en si trabajan juntamente con un fin especifico.
  • Identificar principales herramientas que ofrece GITHUB.
